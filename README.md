# FacultyFlow

FacultyFlow is a React + TypeScript + Tailwind prototype for an investor-ready educational platform focused on helping professors structure course content, manage RAG-backed knowledge, identify weak points, and improve teaching clarity.

## Included

- Responsive dashboard shell with role switching for professor, TA, and admin views
- Course creation and topic organization workflows
- RAG knowledge-base management with approval-focused material review
- Canvas-based concept map, weak-point heatmap, and drag-and-drop lesson planner
- Slide improvement workspace, TA recitation tools, survey trends, and semester planning
- Realistic academic sample data and polished empty states

## Local setup

1. Install Node.js 18+.
2. Install dependencies with `npm install`.
3. Start the app with `npm run dev`.

## Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
